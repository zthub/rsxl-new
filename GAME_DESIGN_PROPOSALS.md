# 弱视训练游戏设计方案

## 一、同时视模块（Simultaneous Perception）

### 游戏名称：红蓝配对消除
**训练原理**：使用红蓝分视技术，训练双眼同时感知不同颜色的图像，建立同时视功能。

**游戏机制**：
- 屏幕上随机出现红色和蓝色的图案（如星星、圆圈、方块等）
- 玩家需要同时看到红色和蓝色图案，点击配对消除
- 只有同时看到两种颜色才能完成配对，单眼看到无法配对
- 随着关卡提升，图案出现速度加快，配对难度增加
- 配对成功得分，配对错误扣分

**训练要点**：
- 强化双眼同时感知能力
- 提高双眼协调性
- 建立稳定的同时视功能

**技术实现**：
- 使用红蓝分视滤镜（CSS filter或Canvas混合模式）
- 红色图案只对左眼可见，蓝色图案只对右眼可见
- 配对时检测是否同时看到两种颜色

## 二、融合视模块（Fusion）

### 游戏名称：融合轨道
**训练原理**：训练双眼将两个略有差异的图像融合成一个，扩大融合范围，消除复视。

**游戏机制**：
- 屏幕中央有一条轨道，轨道上有两个分离的物体（一个红色，一个蓝色）
- 玩家通过鼠标/触摸控制两个物体的位置
- 目标是将两个物体融合成一个（视觉上重叠，形成紫色）
- 融合成功时得分，融合时间越长得分越高
- 如果两个物体分离太远，融合失败
- 随着关卡提升，轨道变复杂，融合难度增加

**训练要点**：
- 扩大融合范围
- 提高融合稳定性
- 消除复视现象
- 建立双眼单视功能

**技术实现**：
- 使用红蓝分视技术，红色物体对左眼可见，蓝色物体对右眼可见
- 当两个物体位置接近时，视觉上会融合成紫色
- 检测融合状态（两个物体距离小于阈值）

## 三、立体视模块（Stereoscopic Vision）

### 游戏名称：深度接球
**训练原理**：利用视差创造深度感，训练立体视觉，建立三维空间深度感知能力。

**游戏机制**：
- 球从屏幕深处（3D空间）向玩家飞来
- 球有不同深度（Z轴位置），玩家需要判断球的深度
- 屏幕底部有一个接球区域，玩家需要移动接球区域到正确位置
- 球越深（Z值越大），在屏幕上显示越小，移动速度越快
- 球越浅（Z值越小），在屏幕上显示越大，移动速度越慢
- 成功接球得分，接球失败扣分
- 随着关卡提升，球的深度变化更复杂，速度更快

**训练要点**：
- 建立三维空间深度感
- 提高立体视觉敏锐度
- 训练深度判断能力
- 强化立体视功能

**技术实现**：
- 使用视差技术，通过左右眼看到略有差异的图像创造深度感
- 计算3D空间中的球的位置，投影到2D屏幕
- 根据深度调整球的大小和移动速度
- 使用红蓝分视或偏振光技术实现立体效果

---

## 游戏设计原则

1. **渐进式难度**：从简单到复杂，逐步提高训练难度
2. **即时反馈**：成功/失败都有明确的视觉和听觉反馈
3. **趣味性**：游戏化设计，提高训练积极性
4. **可量化**：记录训练数据，跟踪训练效果
5. **适配性**：支持不同视力水平的患者，可调节难度

## 技术实现建议

1. **红蓝分视**：使用CSS filter或Canvas混合模式实现
2. **3D效果**：使用视差技术或WebGL实现立体效果
3. **响应式设计**：适配不同屏幕尺寸
4. **性能优化**：使用requestAnimationFrame优化动画性能
5. **数据记录**：记录训练时间、得分、成功率等数据
